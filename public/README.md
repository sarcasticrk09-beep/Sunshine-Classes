# Public Assets Directory

Place your custom logo file here!
To use your custom logo across the entire application:
1. Upload your logo image to this directory.
2. Rename the file to **`logo.png`** (or `logo.jpg`, `logo.jpeg`, `logo.svg`).

The application is configured to automatically detect and display `logo.png` everywhere. If the file is missing or removed, it will seamlessly fall back to the built-in high-resolution vector SVG logo.
